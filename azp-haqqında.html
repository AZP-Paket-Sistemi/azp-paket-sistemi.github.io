AZP Nədir ?

AZP PacPERRO OS tərəfindən hazırlanmış, güvənli və yerli bir paket sistemidir. Paketin uzantısı .azp'dir. Paket sistemi Bash Script ilə yazılmışdır. Bütün Linux distrolarında asanlıqla quraşdırıla bilər və çalışdırıla bilər. Digər paket sistemlərindən fərqli olaraq bağımlılıqlar quraşdırmır. Paket .zip bazalıdır. Hər kəs rahatlıqla öz azp paketini yarada bilər.

Bağımlılıqlar: unzip

$ sudo apt-get install unzip

*Əgər yaratdığınız və düzgün çalışan bir AZP paketiniz varsa PacPERRO Telegram Grupu ilə əlaqəyə keçə bilər, programınızı AZP Deposuna əlavə edə bilərsiniz.
