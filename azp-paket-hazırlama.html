AZP Paketi Hazırlama

AZP Paketi necə hazırlanır ?

1) İlk öncə bir qovluq yaradın (adı fərq etmir).

2) İçinə 3 dənə lazımlı fayl qoymalısınız: app-name, fayllar, info

3) İlk öncə "app-name" faylını yazmalısınız. Faylın içinə sadəcə program adını yazın (program adı böyük hərflərlə olmamalıdır). Bu vacibdir. Məsələn:

4) Daha sonra isə "info" faylını doldurun. Program haqqında məlumatları içinə yazmalısınız. Aşağıdan nümunə götürə bilərsiniz:

"info" faylının içinə yazacağınız başlıqlar bunlardır:

name

description

version

maintainer

(Bunlar standartdır bunlardan başqa başlıqlar yazmayın)

5) İndi isə "fayllar" adlı faylı yazmalısınız. Əslində bu faylı yazaraq paketinizin "src" paketində olan faylların listəsini tutacaqsınız.


6) Bunları tamamladıqdan sonra isə fayl sisteminə uyğun şəkildə qaynaq kodları yerləşdirin və onları src.tar.bz2 faylına sıxışdırın

7) Yuxarıdakı bütün proseslərə əməl etdiyinizdən əminəm. İndi isə layihənizi .zip faylına sıxışdırın.

8) Və son olaraq faylın adını .azp ilə dəyişdirin.
